   -- Submitting Orders --

Your orders should have the following format

#GALAXY <game name> <race name> <password>
<orders>
#END

Send your orders To: slothoub@cs.utwente.nl
with as          Subject: <game name> orders 

The checker is case sensitive so make sure you spell the name with
capitals in the correct places.  

For each turn you submit in as many orders sets as you think necessary,
the GalaxyNG Engine will use the last set at turn time.

  -- Advance Orders --

To send in advance orders add the turn number to the subject line.
So for instance to send in orders for turn 105 for the game Zeal use
as subject Zeal orders 105

command

#REPORT <game name> <race name> <password>

Again    To: slothoub@cs.utwente.nl
with as  Subject:  <game name> orders 

It only works if you have set a password.
